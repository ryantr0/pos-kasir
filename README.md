# 🏪 WARUNG RZ - POS SYSTEM & INVENTORY MANAGEMENT

Sistem Kasir (Point of Sale) modern berbasis Web yang dirancang khusus untuk efisiensi operasional warung dan toko retail. Dilengkapi dengan fitur cerdas berbasis AI dan sistem keamanan multi-role.

---

## ✨ Fitur Utama

* **🔐 Secure Google Auth:** Login cepat dan aman menggunakan integrasi Google OAuth 2.0.
* **📊 Smart Dashboard:** Visualisasi data penjualan real-time dengan grafik statistik yang interaktif.
* **🤖 AI Inventory Assistant:** Terintegrasi dengan **Groq AI** untuk memberikan prediksi stok dan tips manajemen jualan.
* **👥 Multi-Role System:** * **Admin (Owner):** Akses penuh manajemen user dan laporan keuangan.
    * **Kasir:** Fokus pada transaksi penjualan (POS) yang cepat.
* **📦 Inventory Control:** Manajemen produk, kategori, dan notifikasi otomatis jika stok menipis (Radar Restock).
* **📄 Financial Reports:** Export laporan penjualan dan pengeluaran ke format PDF secara instan.

---

## 🛠️ Tech Stack

* **Framework:** [Laravel 11](https://laravel.com)
* **Frontend:** Tailwind CSS (Clean & Bold UI)
* **Database:** SQLite / MySQL
* **AI Integration:** Groq SDK (Llama 3 Model)
* **Authentication:** Laravel Socialite (Google Login)
